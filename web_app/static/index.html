Acerca de

AblacionCorneal

Herramienta clínica para estimar márgenes de seguridad en planificación de ablación corneal, con actualización inmediata de reglas y factor limitante.

Versión web

Usa el mismo core clínico Python que el resto del proyecto y está pensada para consulta rápida en navegador.

Versión desktop

La app nativa para macOS sigue disponible como descarga y se distribuye mediante GitHub Releases.

Herramienta de apoyo clínico. No sustituye el criterio médico ni la decisión asistencial.
